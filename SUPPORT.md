# Support

Adonai is an early open-source alpha.

## For Test Reports

Use the early testing guide:

[docs/early-testing.md](docs/early-testing.md)

Then open an issue with:

- operating system,
- hardware,
- installed inference engines,
- command output,
- what worked,
- what failed,
- what was confusing.

## For Security Issues

Do not open a public issue. Follow [SECURITY.md](SECURITY.md).

## For Product Direction

Use GitHub issues and pull requests for public product discussion.
