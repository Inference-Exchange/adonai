# Code of Conduct

Adonai should be a serious technical community where people can do high-quality work.

## Standard

Be direct, specific, and respectful. Challenge ideas with evidence. Do not attack people.

Unacceptable behavior includes harassment, threats, sexualized language or imagery, sustained disruption, and publishing private information without permission.

## Scope

This applies in GitHub issues, pull requests, discussions, community chats, events, and any project space representing Adonai.

## Enforcement

Maintainers may remove comments, close issues, block users, or take other action needed to protect the project.

Report issues to conduct@compounder.dev.
